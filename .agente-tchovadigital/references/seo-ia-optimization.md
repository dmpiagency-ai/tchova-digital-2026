# Otimização para SEO e IAs (LLMs)

## 🤖 Otimização para IAs (LLM-Friendly)
Para garantir que agentes de IA e crawlers de LLM entendam perfeitamente a página:
- **llms.txt**: Criar um arquivo `/llms.txt` na raiz com o resumo estruturado do serviço.
- **Semantic HTML**: Uso rigoroso de tags `<main>`, `<section>`, `<article>`, `<aside>` e a especificação de `lang="pt-MZ"` na tag `<html>`.
- **Data Attributes**: Usar `data-ai-component="hero"` para facilitar o mapeamento por IAs.

## 🔍 SEO Técnico Avançado
- **JSON-LD**: Implementar Schema.org detalhado.
- **Meta Tags Dinâmicas**: Títulos e descrições otimizados para CTR.
- **Open Graph**: Imagens de 1200x630px com branding visível.

## ⚡ Performance (Core Web Vitals)
- **LCP (Largest Contentful Paint)**: < 2.5s.
- **CLS (Cumulative Layout Shift)**: < 0.1.
- **INP (Interaction to Next Paint)**: < 200ms.
