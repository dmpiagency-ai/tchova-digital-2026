# Guia Mestre: Ecossistema Tchova Digital 2026

## 🎯 1. Visão Geral e Objetivo
O Tchova Digital é um acelerador de autoridade e lucro focado em eliminar o amadorismo visual e operacional.
- **Diferencial**: Hyper-Velocity (entrega 20x mais rápida via IA + Elite Criativa).
- **Tagline**: "Design que Marca & Identidade Visual" / "Soluções de Elite".

## 👥 2. Público-Alvo
1. **Empreendedores e PMEs**: Landing Pages, Branding, Gestão de Anúncios.
2. **Profissionais GSM (GSM Elite)**: Técnicos de reparação mobile (UnlockTool, Chimera, IMEI).
3. **Corporativo/Eventos**: Audiovisual premium (Foto/Vídeo 4K).

## 🛠️ 3. Ecossistema de Serviços
- **Agência Criativa**: Identidade Visual (DNA Visual), Landing Pages (Império Digital), Tráfego Pago (Meta/Google Ads).
- **GSM Tech Hub**: Aluguer de ferramentas virtuais, consultas IMEI, gestão de créditos MZN (Metical Moçambicano).

## 📦 4. Planos Estratégicos
- **ACCELERA™ BRANDING (3.500 MZN (Metical Moçambicano))**: ID Visual em 48-72h.
- **MÁQUINA DE LEADS™ (7.500 MZN (Metical Moçambicano))**: Branding + Landing Page + Anúncios.
- **IMPÉRIO DIGITAL 360™ (15.000 MZN (Metical Moçambicano))**: E-commerce + Pagamentos + IA Atendimento.

## 🧪 5. Stack Tecnológico
- **Frontend**: React 18, Vite, Next.js 15 (App Router).
- **Estilização**: Tailwind CSS v4.
- **Animações**: GSAP (ScrollTrigger, useGSAP), animejs v4.
- **UI/UX**: Premium Dark Mode, Glassmorphism, Fluid Nav.
- **Integrações**: WhatsApp-First, Firebase, GA4, Meta Pixel.
